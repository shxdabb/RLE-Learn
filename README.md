# RLE-Learn

This project contains a simple `GridWorld` environment and a starter Q-learning agent.

## Structure

- `env/gridworld.py`: GridWorld environment implementation.
- `agent/train.py`: Q-learning agent and training loop.

## Usage

Run the environment demonstration:

```bash
python3 env/gridworld.py
```

Train and test the Q-learning agent:

```bash
python3 agent/train.py
```
