RLE-Learn 🚀
A simple beginner-friendly project to understand Reinforcement Learning (RL) using a GridWorld environment and a Q-learning agent.
📌 What this project does

Simulates a small grid-based environment
Trains an agent to reach a goal using trial and error
Demonstrates basic concepts of Q-learning

🧠 Concepts covered

Environment (GridWorld)
Actions (up, down, left, right)
Rewards and penalties
Q-table learning
Exploration vs Exploitation
